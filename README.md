## Descripción

**Editor Markdown** es una aplicación web creada con **Next.js**, **React** y **TailwindCSS** que te permite escribir y visualizar archivos Markdown en tiempo real.

---

## Características

- **Edición en tiempo real:** visualiza los cambios de tu texto al instante.
- **Interfaz moderna:** construida con TailwindCSS para una experiencia fluida y atractiva.
- **Modo oscuro integrado:** diseño adaptable y legible en cualquier entorno.
- **Código limpio y modular:** fácil de entender, mantener y extender.
- **Basado en comunidad:** abierto a mejoras, ideas y nuevas funciones.

---

## Tecnologías utilizadas

- [Next.js 14](https://nextjs.org/)

## Instalación y configuración

### Clonar el repositorio

```bash
git clone git@github.com:CRISHFAS/Editor-Markdown.git
```

### Instalar dependencias

```bash
cd Editor-Markdown
npm install
```

### Iniciar el servidor de desarrollo

```bash
npm run dev
```

### El proyecto se ejecutará en:

```arduino
http://localhost:3000
```
